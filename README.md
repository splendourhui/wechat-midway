# wechat-midway

基于 Egg.js 构建的微信中间服务，用于连接业务和微信

## QuickStart

### Development

```bash
$ npm i
$ npm run dev
$ open http://localhost:3456/
```

### Deploy

```bash
$ npm run tsc
$ npm start
```

### Npm Scripts

- Use `npm run lint` to check code style
- Use `npm test` to run unit test
- se `npm run clean` to clean compiled js at development mode once

### Requirement

- Node.js 8.x
- Typescript 2.8+
